# ternary_operator_in_python3

Python3에서 3항 연산자는 다음과 같이 사용할 수 있습니다.

```
# expression if condition else else_expression
a = 3 if 1 != 2 else 5
print(a) # 3
```

기존의 C 스타일의 3항 연산자는
```
# a = condition ? if_statement : else_statement
a = 1!=2 ? 3 : 5
```
와 같이 할 수 있지만,

Python에서는 C 스타일 보다는 다른 방식을 선호하네요..
저는 저게 더 익숙하지는 않지만..
