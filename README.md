## TIL 작성 규칙
- 문서 생성은 [GFM (Github Flavored Markdown)](https://help.github.com/articles/github-flavored-markdown/) 을 사용한다. (확장자 `.md`)
- 언어나 기술명으로 폴더를 만든다. (root에 문서를 만들지 않는다.)
- 파일명은 영어로.

## Commit Format
- 커밋은 [주제] 커밋 내용 으로 작성합니다.
- 예제) [R] Spearman correlation 구하는 방법
